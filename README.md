# Smart Campus Facility Finder

A cross-platform mobile application developed using React Native and Expo to help students locate university buildings, classrooms, and campus facilities through an interactive map-based interface.

---

## Features

- Interactive campus map
- Building markers and navigation
- Classroom and floor information
- Zoom in and zoom out functionality
- Cross-platform mobile support (Android & iOS)

---

## Technologies Used

- React Native
- JavaScript
- Expo
- React Native Maps

---

## Project Purpose

The application was developed as a final year Software Engineering project at Edge Hill University.

The system was designed to improve campus navigation and reduce confusion when locating university buildings and classrooms.
